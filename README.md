# Blender_addons_leshy
## 123 Select

- Переключение граней вершин в объектном режиме осуществляется простым нажатием клавиш 1, 2, 3.
- Примечание: это не будет работать, если в настройках включена опция «Эмулировать цифровую клавиатуру».
