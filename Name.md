# 🔴Основные префиксы
- `SM_` — Static Mesh (Статик меш, 3D модель)
- `SK_` — Skeletal Mesh (Скелетал меш, анимационная модель)
- `M_` — Material (Материал)
- `MI_` — Material Instance (Экземпляр материала)
- `T_` — Texture (Текстура)
- `BP_` — Blueprint (Блупринт/актор)
- `AS_` — Animation Sequence (основные анимации)
- `PHYS_ `— Physics Asset (Физический ассет) 
# Правила оформления 
- **Структура имени:** `SM_PropName_Var`
- **Пример:** `SM_Chair_01, SM_Door_Wooden_02`
- **Названия только на английском**
